# Power BI Sales Analytics Dashboard

## 📊 Project Overview
This Power BI project provides a complete sales analytics dashboard designed to monitor business performance, sales trends, profitability, and regional insights. The dashboard helps users make data-driven decisions by visualizing important KPIs and business metrics in an interactive format.

---

## 🚀 Features

### 1. Main Overview Dashboard
- Total Sales Analysis
- Profit Tracking
- Quantity Sold Metrics
- Regional Performance Insights
- Interactive Filters and Slicers
- KPI Cards and Trend Visualizations

### 2. Sales Details Dashboard
- Detailed Transaction-Level Analysis
- Product-wise Performance
- Customer and Region Breakdown
- Revenue and Profit Comparisons
- Dynamic Data Exploration

### 3. Tooltip Region Dashboard
- Custom Tooltip Pages
- Enhanced User Interaction
- Region-Specific Insights
